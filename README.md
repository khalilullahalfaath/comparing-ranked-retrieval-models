# Comparing Ranked Retrieval Models

## Overview

This assignment is from the course MII226502 Information Retrieval System (or Sistem Temu Balik Informasi), Master's of Computer Science, Universitas Gadjah Mada, lectured by Yunita Sari, Ph.D. It compares different ranked retrieval models for information retrieval tasks.

## Description

Implement and evaluate various ranking algorithms used in search and retrieval systems, such as TF-IDF, Word2vec, and BERT-based models. This project use news datasets and evaluate it using qualitative metrics as limited in the assignment description on eLOK UGM.

## Requirements

- Python 3.8+
- Required libraries listed in `requirements.txt`

## Installation

```bash
git clone <repository-url>
cd comparing-ranked-retrieval-models
pip install -r requirements.txt
```

## Usage

```bash
python main.py --model <model_name> --dataset <dataset_path>
```
